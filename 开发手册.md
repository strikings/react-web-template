#### 项目开发手册：

> 1. 首先在`GitHub`上新建一个仓库，例如：test-template
>
> 2. 在桌面新建一个test文件夹，在 test 文件夹上（按住shift + 鼠标右键），打开命令窗口
>
>    ```
>    git clone 仓库地址
>    cd 项目目录    		例如：cd test-template
>    ```
>
> 3. 在项目`test-template`根目录运行，如下命令：
>
>    ```
>    git   status   查看状态
>    git   add .     将所有内容放在暂存区
>    git   commit -m "提示语"
>    git   push     上传代码到git仓库
>    ```
